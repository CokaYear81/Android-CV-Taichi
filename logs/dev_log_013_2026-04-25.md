## 当前使用方式

### 处理 raw JSON
1. 将安卓导出的原始 JSON 放入：
   - `data/raw/`
2. 打开：
   - `code/pose17_pipeline.ipynb`
3. 修改配置区：
   - `RAW_JSON_FILENAME`
   - `OUTPUT_LABEL_FOLDER`
4. 运行清洗 cell。
5. 生成的 CSV 会保存到：
   - `data/<OUTPUT_LABEL_FOLDER>/<sample_id>_normalized.csv`

### 可视化清洗后 CSV
1. 将 `VISUALIZE_CSV_PATH` 改成要检查的 CSV 完整路径。
2. 运行读取 CSV 的 cell。
3. 运行静态预览或动画预览 cell。
4. 如果方向不对，修改：
   - `CSV_ROTATION = "clockwise_90"`
   - 或 `counterclockwise_90`
   - 或 `rotate_180`
